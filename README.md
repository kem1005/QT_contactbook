# QT_contactbook

一個使用 Qt 開發的通訊錄管理系統。

## 功能

- **新增聯絡人**：手動輸入學號、班級、姓名和電話來新增聯絡人
- **匯出聯絡人**：將所有聯絡人匯出到 txt 檔案（CSV 格式）
- **匯入聯絡人**：從 txt 檔案匯入聯絡人資料
- **結束**：儲存並關閉程式

## 匯入檔案格式

匯入的 txt 檔案應該使用逗號分隔（CSV 格式），每行包含四個欄位：

```
學號,班級,姓名,電話,
```

範例檔案內容：
```
B10912001,資工一,王小明,0912345678,
B10912002,資工一,李小華,0923456789,
B10912003,資工二,張小美,0934567890,
```

請參考 `sample_import.txt` 檔案作為範例。

## 使用說明

1. 開啟應用程式
2. 使用「新增」按鈕手動新增聯絡人，或使用「匯入」按鈕從檔案載入聯絡人
3. 使用「匯出」按鈕儲存所有聯絡人到檔案
4. 使用「結束」按鈕關閉程式（會自動匯出資料）
